# Arrays

## What is an Array?
An array stores homogeneous datatypes in contigous manner.


Every element will be accessed with an index. 

## Why Arrays?
- The arrays are used for fast index-based access.
- They are used for simple data organization.

## Common Operations
- Traversal
- Insertion
- Deletion
- Searching

## Time Complexity
- Access: O(1)
- Search: O(n)
